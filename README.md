## Setup

### 1. Clone the repo

```bash
git clone <repo-url>
cd <repo-name>
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

Requires Python 3.9+ and a CUDA-capable GPU (tested on RTX 3050, 4 GB VRAM).

### 3. Download the dataset

Download **SDNET2018** from:
> https://digitalcommons.usu.edu/all_datasets/48/

Extract and place the dataset folder at the project root so the structure is:

```
SDNET2018/
  D/   # bridge deck images
  P/   # pavement images
  W/   # wall images
```

## Reproducing the Best Model

To evaluate without retraining, run only **Sections 1–4 and 7** of `10_protonet_final.ipynb` (imports, data splits, model definition, load checkpoint). The checkpoint `checkpoints/proto_final.pt` is included in the repo.

```python
import torch
ckpt = torch.load('checkpoints/proto_final.pt', map_location='cpu')
# emb_dim=256, freeze_until='layer3', preprocessing='clahe', n_support=10
```

All figures in the report are regenerated by Section 7 onwards in `10_protonet_final.ipynb`.

## Note on Randomness

All notebooks use `SEED = 42`. Results may vary slightly across runs due to CUDA non-determinism. To get exact reproducibility, set:

```python
torch.backends.cudnn.deterministic = True
torch.backends.cudnn.benchmark = False
```
